# Guide d'installation

## Pourquoi il n'y a pas de `setup.exe` ?

Par mesure de sécurité, aucun installateur (`setup.exe`) n'est fourni.

L'objectif est que vous sachiez exactement ce que vous téléchargez et exécutez sur votre ordinateur, sans installation cachée ni modification automatique de votre système.

## Installation

1. Téléchargez tous les fichiers **.zip** du projet.
2. Téléchargez également le fichier **.exe** disponible dans l'onglet **Releases**.
3. Décompressez les fichiers **.zip**.
4. Placez le fichier **.exe** à la racine du projet (dans le même dossier que les autres fichiers du projet).
5. Lancez l'exécutable.

L'application est maintenant prête à être utilisée.
